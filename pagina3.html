Regras CSS

uma regra css pode ser definida com a seguinte sintaxe

        Seletor{
            Propriedade1:Valor
            Propriedade2:Valor

            PropriedadesN:Valor
        }

Pode ser inciado de três formas diferentes

CSS Externo
Quando definido as regras de CSS em um arquivo externo (normalmente style.css), vinculado por meio da tag link no head do doc
CSS Incorporado
Regras style css feitas direto no html antes do body
CSS Inline
Quando definido o código css style no próprio URL como: Texto de cor vermelha com CSS Inline
